# AI Image Caption Generator - Frontend

This directory contains the React-based frontend for the AI Image Caption Generator.

## Getting Started

To get the application running locally, follow these steps:

### 1. Navigate to the Frontend Directory

Open your terminal and change your directory to the `frontend` folder:

```bash
cd frontend
```

### 2. Install Dependencies

Install the necessary Node.js packages using `npm`:

```bash
npm install
```

### 3. Start the Development Server

Once the dependencies are installed, you can start the React development server:

```bash
npm start
```

This will open a new tab in your web browser with the application running at [http://localhost:3000](http://localhost:3000).

## Project Structure

-   `public/index.html`: The main HTML file for the application.
-   `src/index.js`: The entry point of the React application.
-   `src/App.js`: The main application component containing the UI structure.
-   `src/App.css`: The stylesheet for the main application component, defining the professional, corporate aesthetic.
-   `package.json`: Defines the project's dependencies and scripts.
