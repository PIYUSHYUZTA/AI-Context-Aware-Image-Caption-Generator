                            DATA FLOW DIAGRAM

┌──────────────┐
│   USER       │
│   UPLOADS    │
│   IMAGE      │
└──────┬───────┘
       │
       ▼
┌──────────────────────────────────────────────────────────────┐
│                    IMAGE PREPROCESSING                       │
│  • Resize to 224x224                                         │
│  • Convert to array                                          │
│  • Normalize values                                          │
└──────────────────────┬───────────────────────────────────────┘
                       │
                       ▼
┌──────────────────────────────────────────────────────────────┐
│                  FEATURE EXTRACTION (VGG16)                  │
│  • Pass through convolutional layers                         │
│  • Extract 4096-dimensional feature vector                   │
│  • Represents visual content                                 │
└──────────────────────┬───────────────────────────────────────┘
                       │
                       ▼
┌──────────────────────────────────────────────────────────────┐
│                  CAPTION GENERATION (LSTM)                   │
│  1. Start with 'startseq' token                              │
│  2. Combine features + previous words                        │
│  3. Predict next word                                        │
│  4. Add to sequence                                          │
│  5. Repeat until 'endseq' or max length                      │
└──────────────────────┬───────────────────────────────────────┘
                       │
                       ▼
┌──────────────────────────────────────────────────────────────┐
│                    POST-PROCESSING                           │
│  • Remove start/end tokens                                   │
│  • Clean up caption                                          │
│  • Calculate metrics                                         │
└──────────────────────┬───────────────────────────────────────┘
                       │
                       ▼
┌──────────────────────────────────────────────────────────────┐
│                    DISPLAY RESULTS                           │
│  • Show generated caption                                    │
│  • Display performance metrics                               │
│  • Provide analysis                                          │
└──────────────────────────────────────────────────────────────┘

                            QUICK REFERENCE

Launch Application:
  streamlit run app_eduvision.py

Check Tokenizer:
  python -c "from pickle import load; t = load(open('tokenizer.pkl', 'rb')); 
  print(f'Vocab: {len(t.word_index)+1}')"

Retrain Model:
  python train_improved.py

Verify Setup:
  python check_model.py
